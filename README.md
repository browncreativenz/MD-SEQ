# MD-SEQ
Generative performance sequencer for Elektron Machinedrum and Norns.

MD-SEQ is a generative sequencer built for Norns + Grid, designed to explore evolving rhythmic structures and real-time pattern mutation on the Elektron Machinedrum. It combines hands-on randomness controls, pattern saving, and live performance tools â€” all in a minimal, responsive interface.
